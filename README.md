#SnowBall Stemmer<br>
Snowball is a small string processing language designed for creating stemming algorithms for use in Information Retrieval. <br>
The Snowball stemmer can be found here. http://snowball.tartarus.org
## Supported Languages
- danish
- dutch
- english
- finnish
- german
- hungarian
- italian
- norwegian
- romanian
- russian
- spanish
- swedish
- porter
- portuguese
- romanian
- spanish
- turkish
